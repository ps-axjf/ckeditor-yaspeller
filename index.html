Русский

About

CKeditor-yaspeller is plugin for CKEditor for spell-checking Russian and English languages while you type. Mispelled words are marked, and you just need to right-click mispelled word to select a suggestion to replace the word with. You can change errors markup style using CSS, and enable/disable spell-checking in current editor.

Demo

Mark errors as:
  • red font
  • blue font and yellow background
  • black font and red underline
  • or you can use your own css

You can download this project in either zip or tar formats. Or visit our public github repository.

English

Описание

CKeditor-yaspeller это плагин для CKEditor который проверяет грамматические ошибки на русском и английском языках. Ошибочные слова выделяются, и всё что нужно это кликнуть по такому слову и выбрать правильное слово из предлагаемых замен. Вы можете менять стиль выделения ошибочных слов, и включать/выключать проверку в текущем редакторе.

Демо

Выделять ошибки:
  • красным цветом
  • синим цветом и желтвым фоном
  • красным подчеркиванием
  • или использовать собственный стиль
